## 📌 Project Overview

The **Vrinda Store Data Analysis Dashboard** is an interactive Business Intelligence project developed using **Microsoft Power BI**. The dashboard transforms raw sales data into meaningful insights, enabling better business decisions through interactive visualizations.

This project analyzes sales performance, customer demographics, order status, regional sales, and sales channels to identify key business trends.

# 📈 Key Business Insights

* Women customers contribute a significant portion of total sales.
* Adult customers generate the highest number of purchases.
* Amazon is one of the top-performing sales channels.
* Maharashtra, Karnataka, and Uttar Pradesh contribute significantly to sales.
* Monthly trend analysis helps identify seasonal demand.
* Most orders are successfully delivered, with a smaller percentage of returns and cancellations.
